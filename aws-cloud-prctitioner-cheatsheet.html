Critical Exam Tips — Things to Know Cold

  • S3 is NOT a file system — it's object storage
  • EBS = single AZ; EFS = multi-AZ file system
  • Glacier is for archives; Standard has 11 9s durability
  • CloudFront caches; Global Accelerator routes (no cache)
  • NAT Gateway = outbound only for private subnets
  • Security Groups = stateful; NACLs = stateless
  • IAM is global — not region-specific
  • Root account = disable & lock down immediately
  • Spot Instances = cheapest; can be interrupted
  • Reserved = 1 or 3 year commit; All Upfront = max savings
  • CloudTrail = API audit log (compliance)
  • CloudWatch = metrics & alarms (monitoring)
  • GuardDuty = threat detection (ML)
  • Inspector = vulnerability scanning (EC2/containers)
  • Macie = sensitive data discovery (S3 PII)
  • SQS = pull-based queue; SNS = push pub/sub
  • Direct Connect > VPN for bandwidth & consistency
  • Business Support = minimum for Trusted Advisor all checks
  • AWS Artifact = compliance reports (not Audit Manager)
  • Concierge support = Enterprise & Enterprise On-Ramp only
